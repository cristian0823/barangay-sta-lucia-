System Updates: Added Features & Enhancements

•    Dark Mode: Implemented a comprehensive dark mode theme throughout the admin dashboard to ensure a comfortable viewing experience and consistent styling across all interface elements.
•    Activity Log: Added an automated activity tracking system that records significant actions securely behind the scenes (like user registrations), maintaining transparency for admin monitoring.
•    Report: Developed dedicated reporting capabilities that allow administrators to view, track, and export structured data summaries of system traffic and historical logs.
•    Overview: Refined the main dashboard overview panel by replacing static content with a dynamic "Recent Activity" feed, giving admins a live, real-time snapshot of system updates the moment they log in.
